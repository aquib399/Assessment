# Internshala Assessment

A Node.Js and Express.Js application

## Installation

Use the NPM to install express, mongodb

```bash
npm install express mongodb
```

## Usage

```bash
npm start
```

#### Development mode (Nodemon)

```bash
npm run dev
```
